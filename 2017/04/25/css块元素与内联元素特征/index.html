css块元素与内联元素

block块元素特点：

  • 重新开始一行
  • 宽度可设定
  • 行高及顶和底边距不可改变

inline元素特点：

  • 和其他元素在一行
  • 宽度为文字或图片宽度不可变（img、input除外）
  • 高，行高及顶和底边距不可改变
    块元素与内联元素的嵌套规则：块元素可以包含内联元素或某些块元素，但内联元素却不能包含块元素，它只是包含其他的内联元素。

块元素分类

-body
-可嵌套元素
-不可嵌套元素(h1-h6、caption、p、hr、dt）

HTML里有几个元素是比较特别的：ul、ol、dl、table，它们的子一层必须是指定元素，ul、ol的子一级必须是li；dl的子一级必须是dt或者dd；table的子一层必须是caption 或thead、tfoot、tbody等，而再子一层必须是tr （tr只存在于thead、tfoot、tbody中），之后才是可放内容的td或者th。
